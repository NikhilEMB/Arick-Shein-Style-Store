Creating Template(s)

** You have to create the following 4 template(s) for order notifications on the interakt dashboard and get them approved.
** Set the display name accordingly as provided below.
** Set the category as " Account Update ".
** Set the language as only " English ".

** Copy paste the following template(s) as the template body in the respective template(s).

1. Display Name - order_confirmed Template Body 
Hi {{templateVal1}},
                  
ORDER CONFIRMED ✅
                  
Thank you for your order. Your Order id is {{templateVal2}}.
                  
We will inform you when we dispatch your order.
                  
{{templateVal3}}
2. Display Name - order_dispatched Template Body 
Hi {{templateVal1}}, 

ORDER DISPATCHED 🚀 
                    
Your order with Order id {{templateVal2}} is dispatched and will reach to you soon. 
                    
{{templateVal3}}
3. Display Name - order_cancelled Template Body 
Hi {{templateVal1}},

ORDER CANCELLED ❌ 
                    
Your order with Order id {{templateVal2}} is cancelled. 
                    
{{templateVal3}}
4. Display Name - order_delivered Template Body 
Hi {{templateVal1}},

ORDER DELIVERED 🏠
                    
Your order with Order id {{templateVal2}} is delivered at your selected address.
                    
Thank you for shopping with us.
                    
{{templateVal3}}

** Specified Values :

{{templateVal1}} - The Users name receiving the order.
{{templateVal2}} - The order ID for the specified order.
{{templateVal3}} - The Store name
